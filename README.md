# Dashboard-examples
Examples of my dashboards in Tableau

Coffee shop sales
Dashboard includes 2 pages:    
1. "Current dymanics" - KPIs of Coffee shop with filters by period, drinks, weekdays, time of day.
2. "Monthly dynamics" - Monthly KPIs by years, drinks, weekdays, time of day.
You can chose different metrics on the charts: sum sales, avg sales, median sales.
